# Kevin Iris Fabien

## Usage
Simply clone the repository :
```
git clone https://github.com/simplon-CQP-Java/kif.git && cd kif
```

Maven clean
```
mvn clean
```

Launch the app
```
mvn tomcat7:run
```

App running by default to port 8080
So open your browser and go to ```http://localhost:8080/```
